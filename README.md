# Amatrans Website Clone

This is a clone of the [Amatrans.it](https://www.amatrans.it/) website, created using HTML, CSS, and JavaScript.

## Features

- Pixel-perfect recreation of the Amatrans.it website
- Responsive design that works on mobile, tablet, and desktop screens
- Dark blue gradient background with a professional shipping company aesthetic
- Interactive contact buttons with hover effects
- Smooth animations when scrolling
- Multi-language content display (Italian, English, Turkish)

## Files

- `index.html` - The main HTML structure of the website
- `styles.css` - All styling for the website, including the blue gradient background and responsive design
- `script.js` - JavaScript functionality for animations and interactive elements

## How to Use

1. Open the `index.html` file in a web browser to view the website
2. Resize the browser window to see the responsive design in action
3. Hover over the contact buttons to see the interactive effects
